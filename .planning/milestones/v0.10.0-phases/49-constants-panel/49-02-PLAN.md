# Plan 49-02: Inspector Tab System and Constants Panel UI

<objective>
Add a tab system to the existing inspector pane (Node tab + Constants tab) and create the constants panel UI with grouping, search/filter, number input + slider editing, per-constant reset, global Reset All, flash animation on edit, and modified-value indicator.
</objective>

2. **Add tab bar renderer:**
```typescript
function renderTabBar(): string {
  const nodeActive = _activeTab === 'node';
  const constActive = _activeTab === 'constants';
  // Active tab: color #f0f6ff, border-bottom 2px solid #00f0ff
  // Inactive tab: color #7981a0, no border-bottom, hover color #a5abc2
  return `
    <div id="inspector-tab-bar" style="display:flex;border-bottom:1px solid #1f2235;padding:0 12px;gap:0;flex-shrink:0">
      <button data-inspector-tab="node"
        style="padding:8px 14px;font-size:10px;letter-spacing:0.15em;text-transform:uppercase;font-family:inherit;
               background:transparent;border:none;cursor:pointer;
               color:${nodeActive ? '#f0f6ff' : '#7981a0'};
               border-bottom:${nodeActive ? '2px solid #00f0ff' : '2px solid transparent'}">
        Node
      </button>
      <button data-inspector-tab="constants"
        style="padding:8px 14px;font-size:10px;letter-spacing:0.15em;text-transform:uppercase;font-family:inherit;
               background:transparent;border:none;cursor:pointer;
               color:${constActive ? '#f0f6ff' : '#7981a0'};
               border-bottom:${constActive ? '2px solid #00f0ff' : '2px solid transparent'}">
        Constants
      </button>
    </div>
  `;
}
```

3. **Modify `updateInspectorPane()`** to render tab bar + active tab content:
- Tab bar is always shown when inspector is open (even with no node selected)
- When `_activeTab === 'node'`: render the existing node detail content (or empty state if no node selected)
- When `_activeTab === 'constants'`: call `renderConstantsPanel()` from `constants-panel.ts`

4. **Modify `showInspector()`** to set `_activeTab = 'node'` when a node is clicked.

5. **Add `openConstantsTab()` export** that opens the inspector with the constants tab active (no node selection required).

6. **Wire tab click handlers** inside `updateInspectorPane()`:
- After setting innerHTML, add event listeners on `[data-inspector-tab]` buttons
- Tab click sets `_activeTab` and calls `updateInspectorPane()`

7. **Modify `renderEmptyInspector()`** to show tab bar + empty node state, so the constants tab is accessible even when no node is selected.

8. **Export `getActiveTab()` and `setActiveTab(tab: InspectorTab)`** for external access.

9. **When inspector is hidden (width 0), opening constants tab should also trigger inspector-open** class on `#flow-area-inner`.
</action>

Create `apps/demo/src/constants-panel.ts` with:

1. **Module state:**
```typescript
let _groupingMode: 'package' | 'domain' | 'flat' = 'package';
let _searchQuery = '';
```

2. **`renderConstantsPanel()` function** (exported, returns HTML string):

**Header section:**
- Title: "protocol constants" (10px uppercase, same style as inspector sections)
- Reset All button: `<button id="constants-reset-all" ...>Reset All</button>` — visible only when any constant is modified. Style: transparent bg, #ff3b3b border, #ff3b3b text, small (10px)
- Search input: `<input id="constants-search" type="text" placeholder="filter..." />` — full width, dark bg (#0a0a0f), border #3a3a4a, 11px

**Grouping mode selector:**
- Three small toggle buttons: "package" | "domain" | "flat"
- Active mode: bg rgba(0,240,255,0.1), border #00f0ff, color #00f0ff
- Inactive: transparent, border #3a3a4a, color #7981a0

**Constants list:**
- Get grouped defs from `demoConfig.getByPackage()`, `demoConfig.getByDomain()`, or flat `demoConfig.getAllDefs()` based on `_groupingMode`
- Filter by `_searchQuery` (match against key, label, description — case-insensitive)
- Each group has a header: group name in 10px uppercase, color #7c86a7, letter-spacing 0.2em

**Per-constant row:**
```html
<div class="const-row" data-const-key="{key}" style="padding:6px 0;border-bottom:1px solid #1a1a28;position:relative">
  <!-- Modified indicator: small dot, visible only when isModified -->
  <div style="position:absolute;left:-8px;top:50%;transform:translateY(-50%);width:4px;height:4px;border-radius:50%;background:#00f0ff;display:{isModified ? 'block' : 'none'}"></div>
  
  <!-- Label row -->
  <div style="display:flex;justify-content:space-between;align-items:center;margin-bottom:4px">
    <span style="font-size:10px;color:#d0d4e8">{label}</span>
    <span style="font-size:9px;color:#666">{unit}</span>
  </div>
  
  <!-- Read-only display (for protocol constants): -->
  {if !editable}
    <div style="font-size:12px;color:#62d0ff;padding:2px 0">{currentValue}</div>
  {/if}
  
  <!-- Editable controls: number input + slider (for behavioral constants): -->
  {if editable}
    <div style="display:flex;gap:8px;align-items:center">
      <input type="number" data-const-input="{key}" 
        value="{currentValue}" min="{min}" max="{max}" step="{step}"
        style="width:72px;background:#0a0a0f;border:1px solid #3a3a4a;border-radius:4px;padding:3px 6px;color:#f0f6ff;font-family:inherit;font-size:11px;text-align:right" />
      <input type="range" data-const-slider="{key}"
        value="{currentValue}" min="{min}" max="{max}" step="{step}"
        style="flex:1;accent-color:#00f0ff;height:4px" />
      <!-- Reset icon: visible only when modified -->
      <button data-const-reset="{key}" 
        style="background:transparent;border:none;color:#7981a0;cursor:pointer;font-size:12px;padding:2px;display:{isModified ? 'inline' : 'none'}"
        title="Reset to default ({defaultValue})">↺</button>
    </div>
  {/if}
  
  <!-- Description -->
  <div style="font-size:9px;color:#555;margin-top:2px">{description}</div>
</div>
```

3. **`wireConstantsPanelEvents()` function** (exported, called after innerHTML set):

Wire these event handlers via event delegation on the constants panel container:

- **Number input change** (`input` event on `[data-const-input]`): parse value, call `demoConfig.set(key, value)`, update adjacent slider, trigger flash
- **Range slider change** (`input` event on `[data-const-slider]`): parse value, call `demoConfig.set(key, value)`, update adjacent number input, trigger flash
- **Per-constant reset** (`click` on `[data-const-reset]`): call `demoConfig.reset(key)`, update input+slider to default, trigger flash
- **Reset All** (`click` on `#constants-reset-all`): call `demoConfig.resetAll()`, re-render panel
- **Search** (`input` on `#constants-search`): update `_searchQuery`, re-render constants list
- **Grouping toggle** (`click` on `[data-grouping-mode]`): update `_groupingMode`, re-render

4. **Flash animation on edit:**
- When a value changes, add a brief green background pulse to the row:
```typescript
function flashConstantRow(key: string): void {
  const row = document.querySelector(`[data-const-key="${key}"]`) as HTMLElement | null;
  if (!row) return;
  row.style.background = 'rgba(57,255,20,0.08)';
  setTimeout(() => { row.style.background = 'transparent'; }, 400);
}
```

5. **Grouping labels:**
- Package: 'core', 'runtime', 'services', 'acl', 'shell', 'demo'
- Domain: 'timeouts', 'sizes & limits', 'ui timing', 'protocol (read-only)'
</action>
