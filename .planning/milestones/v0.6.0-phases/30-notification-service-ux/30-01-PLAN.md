<interfaces>
### Notification controller shape

```typescript
export interface DemoNotificationSnapshot {
  notifications: readonly Notification[];
  unreadCount: number;
  lastUpdatedAt: number | null;
  sourceLabel: string | null;
}

export interface DemoNotificationController {
  getSnapshot(): DemoNotificationSnapshot;
  subscribe(listener: (snapshot: DemoNotificationSnapshot) => void): () => void;
  createDemoNotification(input: { title: string; body: string; sourceLabel: string }): void;
  requestList(windowId?: string): void;
  markRead(id: string): void;
  dismiss(id: string): void;
  connectRuntime(runtime: ShellBridge['runtime']): void;
}
```

### Service registration target

`createDemoHooks().services` must contain:

```typescript
services: {
  signer: createSignerService({ getSigner: signerHooks.getSigner }),
  notifications: createNotificationService({ onChange: controller.handleChange }),
}
```

The exact callback function name may differ, but the registration must be real and must route changes into the controller.
</interfaces>
