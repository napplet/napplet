<objective>
Add event.stopPropagation() to service button click handlers to prevent bubbling to the node selection handler.

Purpose: Currently, clicking service buttons (signer connect, notification create, etc.) inside node cards triggers both the button action AND the node selection handler, opening the inspector pane unexpectedly. Calling event.stopPropagation() isolates button clicks from parent node listeners.

Output: All service button click handlers in main.ts updated with event.stopPropagation() calls, ensuring buttons execute their actions without triggering node selection.
</objective>
