# Plan 18-03: Shell Migration — Delete Old Service Types, Update Shell Imports

<objective>
Remove ServiceDescriptor, ServiceHandler, and ServiceRegistry definitions from @napplet/shell.
Replace with imports from their new canonical locations (@napplet/core and @napplet/runtime).
Re-export for backwards compatibility. Pass services through hooks-adapter to RuntimeHooks.

This is a hard cut (D-01) — no shim/bridge between old and new types. The old definitions are
deleted and replaced with imports. Shell consumers who import service types from @napplet/shell
continue to work via re-exports.
</objective>
