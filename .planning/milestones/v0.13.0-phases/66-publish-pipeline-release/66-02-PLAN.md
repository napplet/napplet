  <files></files>
  <action>
The user must complete 3 steps that require human authentication:

1. **Create the @napplet npm organization** (if it does not already exist):
   - Go to https://www.npmjs.com/org/create
   - Organization name: `napplet`
   - Choose the free/open-source plan

2. **Generate an npm access token**:
   - Go to https://www.npmjs.com/settings/YOUR_USERNAME/tokens
   - Click "Generate New Token" -> "Classic Token"
   - Select type: **Automation** (this type works in CI without 2FA prompts)
   - Copy the token

3. **Add NPM_TOKEN as a GitHub repository secret**:
   - Go to https://github.com/sandwichfarm/napplet/settings/secrets/actions
   - Click "New repository secret"
   - Name: `NPM_TOKEN`
   - Value: paste the npm token from step 2
  </action>

  <done>npm org exists, NPM_TOKEN GitHub secret is configured</done>
  <resume-signal>Type "done" when the npm org exists, the token is generated, and the GitHub secret is set</resume-signal>
