<objective>
Amend NUB-RELAY with the optional sidecar field on `RelayEventMessage` and wire the relay shim to hydrate the resource shim's single-flight cache before delivering each event to the napplet's `onEvent` callback.

Purpose: Lets shells that opt in (per Phase 132 spec policy) pre-resolve resources referenced by relay events. The napplet's `napplet.resource.bytes(url)` call then resolves from cache without a postMessage round-trip — invisibly to the napplet caller. This phase ships the wire/code only; the default-OFF policy + per-event-kind allowlist guidance live in the NUB-RELAY spec amendment (Phase 132).

Output:
- `packages/nub/src/relay/types.ts` — `RelayEventMessage` extended with optional `resources?: ResourceSidecarEntry[]` field; type-only sibling import of `ResourceSidecarEntry`.
- `packages/nub/src/relay/shim.ts` — inbound `relay.event` handler calls `hydrateResourceCache(msg.resources)` before invoking `onEvent(msg.event)`; runtime sibling import of `hydrateResourceCache` from `../resource/shim.js`.
- Acceptance proven via Node smoke test against built dist: a fake `relay.event` envelope with a non-empty `resources` array, fed through the shim's event handler, leaves the synchronous `bytes(sidecarUrl)` call inside `onEvent` resolving to the same Blob bytes WITHOUT touching `window.parent.postMessage`.
</objective>

```typescript
function handleMessage(msgEvent: MessageEvent): void {
  if (msgEvent.source !== window.parent) return;
  const msg = msgEvent.data;
  if (typeof msg !== 'object' || msg === null || typeof msg.type !== 'string') return;
  if (!msg.type.startsWith('relay.')) return;

  const typedMsg = msg as RelayEventMessage | RelayEoseMessage | RelayClosedMessage;
  if (!('subId' in typedMsg) || typedMsg.subId !== subId) return;

  if (msg.type === 'relay.event') {
    onEvent((msg as RelayEventMessage).event);   // <-- hydrate BEFORE this line
  } else if (msg.type === 'relay.eose') {
    onEose();
  } else if (msg.type === 'relay.closed') {
    window.removeEventListener('message', handleMessage);
  }
}
```
