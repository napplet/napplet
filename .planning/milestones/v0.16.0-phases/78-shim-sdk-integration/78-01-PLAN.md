<objective>
Migrate @napplet/shim from NIP-01 array wire format to JSON envelope messages using NUB module types.

Purpose: This is the core wire format migration that fulfills SHIM-01 (sends JSON envelopes), SHIM-02 (receives JSON envelopes), and SHIM-03 (window.napplet API unchanged). Every outbound postMessage call changes from `['VERB', ...]` arrays to `{ type: 'domain.action', ... }` objects. Every inbound handler switches from array verb dispatch to envelope type dispatch.

Output: All 6 shim source files migrated to JSON envelope format. NUB packages added as dependencies.
</objective>
