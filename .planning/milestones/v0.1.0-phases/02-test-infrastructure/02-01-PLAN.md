<task type="auto">
  <name>Task 2: Create root Vitest configuration</name>
  <files>vitest.config.ts</files>
  <read_first>package.json, turbo.json</read_first>
  <action>
  Create `vitest.config.ts` at the project root with Node-mode configuration:

  ```typescript
  import { defineConfig } from 'vitest/config';

  export default defineConfig({
    test: {
      // Node mode for unit tests (no browser)
      environment: 'node',
      // Include test files from packages and tests directories
      include: [
        'packages/*/src/**/*.test.ts',
        'tests/unit/**/*.test.ts',
      ],
      // Exclude e2e tests (handled by Playwright)
      exclude: [
        'tests/e2e/**',
        'node_modules/**',
      ],
      // Coverage configuration
      coverage: {
        provider: 'v8',
        include: ['packages/*/src/**/*.ts'],
        exclude: ['packages/*/src/**/*.test.ts', 'packages/*/dist/**'],
      },
    },
  });
  ```

  This configuration:
  - Uses Node environment (no jsdom — per PITFALLS.md, jsdom cannot test postMessage protocols)
  - Includes unit tests from all packages and a shared tests/unit/ directory
  - Excludes e2e tests (those use Playwright)
  - V8 coverage provider for code coverage
  </action>
  <verify>test -f vitest.config.ts && grep "defineConfig" vitest.config.ts && grep "environment.*node" vitest.config.ts</verify>
  <acceptance_criteria>
    - vitest.config.ts exists at project root
    - Contains `environment: 'node'`
    - Contains `include` pattern for `packages/*/src/**/*.test.ts`
    - Contains `exclude` pattern for `tests/e2e/**`
    - `npx vitest run --passWithNoTests` exits 0
  </acceptance_criteria>
  <done>Root Vitest config created with Node-mode settings</done>
</task>

<task type="auto">
  <name>Task 4: Create root Playwright configuration</name>
  <files>playwright.config.ts</files>
  <read_first>package.json</read_first>
  <action>
  Create `playwright.config.ts` at the project root:

  ```typescript
  import { defineConfig, devices } from '@playwright/test';

  export default defineConfig({
    testDir: './tests/e2e',
    fullyParallel: true,
    forbidOnly: !!process.env.CI,
    retries: process.env.CI ? 2 : 0,
    workers: process.env.CI ? 1 : undefined,
    reporter: 'list',
    use: {
      baseURL: 'http://localhost:4173',
      trace: 'on-first-retry',
    },
    projects: [
      {
        name: 'chromium',
        use: {
          ...devices['Desktop Chrome'],
          // Use system Chromium — do NOT use bundled browser
          launchOptions: {
            executablePath: '/usr/bin/chromium',
          },
        },
      },
    ],
    // Web server for serving the test harness and napplets
    webServer: {
      command: 'pnpm test:serve',
      url: 'http://localhost:4173',
      reuseExistingServer: !process.env.CI,
      timeout: 30000,
    },
  });
  ```

  Key choices:
  - Uses system Chromium (per AGENTS.md, browsers are installed via pacman)
  - Single browser project (Chromium only, per ROADMAP out-of-scope: "Multi-browser CI matrix")
  - Web server auto-starts via `pnpm test:serve` (to be created in Plan 05)
  - Base URL at port 4173 (Vite preview port)
  </action>
