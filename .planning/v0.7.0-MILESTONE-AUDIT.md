## Critical Gaps (Blockers)

### SESS-03 — loadOrCreateKeypair not renamed ✗

**Requirement:** `loadOrCreateKeypair()` renamed to `createEphemeralKeypair()` — name matches actual behavior (always creates fresh keypair, never loads); unused `_nappType` parameter removed.

**What happened:** Phase 38-02 narrowed scope from rename to TODO-only. The SUMMARY explicitly states: *"function name unchanged"* and *"`grep 'createEphemeralKeypair'` → 0 hits (not renamed)"*.

**Evidence of gap:**
```
packages/shim/src/napplet-keypair.ts:31  export function loadOrCreateKeypair(_nappType: string)
packages/shim/src/index.ts:228           keypair = loadOrCreateKeypair(nappletType)
packages/shim/src/index.ts:344           keypair = loadOrCreateKeypair(nappletType)
packages/shim/src/keyboard-shim.ts:9    // after loadOrCreateKeypair via setKeyboardShimKeypair()
```

**Fix:** Rename function + update 4 call sites + remove `_nappType` parameter.
