# Quick Task 260401-p5s: Make Topology Lines Symmetrical

## Problem

In `apps/demo/src/topology.ts` `initTopologyEdges()`, both the `outLine` and `inLine`
had negative X socket gravity values (`[-20, 0]` and `[-55, 0]`), causing both lines
to curve in the same direction (left). They were also too close together (35px separation).

## Fix

Mirror the gravity values symmetrically and increase magnitude for clear visual separation:

| Line    | Before                  | After                 |
|---------|-------------------------|-----------------------|
| outLine | `[-20, 0]` / `[-20, 0]` | `[+60, 0]` / `[+60, 0]` |
| inLine  | `[-55, 0]` / `[-55, 0]` | `[-60, 0]` / `[-60, 0]` |

## Tasks

### Task 1: Fix gravity values in topology.ts

- **File**: `apps/demo/src/topology.ts` lines ~208-224
- **Action**: Change outLine gravity to `[60, 0]`, inLine gravity to `[-60, 0]`
- **Done**: ✅
