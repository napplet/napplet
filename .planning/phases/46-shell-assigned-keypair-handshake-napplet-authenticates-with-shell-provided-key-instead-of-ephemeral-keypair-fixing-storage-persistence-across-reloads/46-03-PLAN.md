# Plan 46-03: Instance GUID Assignment

## Objective

Implement persistent per-iframe GUID assignment so each napplet iframe slot gets a stable identifier that survives page reloads. The GUID is assigned by the shell/runtime when an iframe first registers and is stored via a persistence hook. Same tab/slot = same GUID across reloads.

<action>
In `packages/runtime/src/types.ts`:

1. Add a GUID persistence adapter after the `ShellSecretPersistence` interface (if Plan 01 was applied, otherwise after `StatePersistence`):

```typescript
/**
 * GUID persistence — runtime calls these to save/load per-iframe instance GUIDs.
 * GUIDs survive page reloads: same iframe slot gets the same GUID.
 * Implementor decides storage backend and keying strategy
 * (e.g., localStorage keyed by iframe src or slot index).
 */
export interface GuidPersistence {
  /** Get a stored GUID for a window identifier, or null if none exists. */
  get(windowId: string): string | null;
  /** Store a GUID for a window identifier. */
  set(windowId: string, guid: string): void;
  /** Remove a stored GUID. */
  remove(windowId: string): void;
}
```

2. Add `instanceId` to the `SessionEntry` interface (around line 344):

In the existing `SessionEntry` interface, add after `registeredAt: number;`:

```typescript
  /** Persistent GUID for this iframe instance, assigned by the runtime. Survives page reloads. */
  instanceId: string;
```

3. Add `guidPersistence` to the `RuntimeAdapter` interface, as an optional field:

```typescript
  /** GUID persistence for iframe instance tracking (optional — if absent, GUIDs are in-memory only). */
  guidPersistence?: GuidPersistence;
```
</action>

<action>
In `packages/runtime/src/session-registry.ts`:

1. Add a `getInstanceId` method to the `SessionRegistry` interface, after `getAllEntries()`:

```typescript
  /** Get the instance GUID for a window. */
  getInstanceId(windowId: string): string | undefined;
```

2. Implement it in `createSessionRegistry()` — the GUID is stored as part of `SessionEntry.instanceId`:

```typescript
    getInstanceId(windowId: string): string | undefined {
      const pubkey = byWindowId.get(windowId);
      if (!pubkey) return undefined;
      return byPubkey.get(pubkey)?.instanceId;
    },
```

This is a simple accessor — the actual GUID assignment happens in the runtime's REGISTER handler (Plan 04).
</action>
