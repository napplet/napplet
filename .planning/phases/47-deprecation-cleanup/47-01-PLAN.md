# Plan 47-01: Remove RuntimeHooks and ShellHooks Deprecated Aliases

## Objective

Delete the `RuntimeHooks` and `ShellHooks` deprecated type aliases and the `createMockRuntimeHooks` deprecated test utility. Update all internal references (imports, call sites, README examples, JSDoc comments) to use the canonical names (`RuntimeAdapter`, `ShellAdapter`, `createMockRuntimeAdapter`). After this plan, importing `RuntimeHooks` from `@napplet/runtime` or `ShellHooks` from `@napplet/shell` produces a compile-time error.
