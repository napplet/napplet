# Plan 50-01: Extend AclCheckEvent and Wire ACL Audit in Demo

<objective>
Extend the AclCheckEvent interface with an optional NIP-01 message field, modify the enforce gate to accept and forward message context, create a per-napplet ACL event ring buffer in the demo, and wire the onAclCheck callback in shell-host.ts so every ACL decision is captured with full event context.
</objective>

<title>Pass message context at enforce() call sites in runtime</title>
<read_first>
- packages/runtime/src/runtime.ts
- packages/runtime/src/enforce.ts
- packages/runtime/src/event-buffer.ts
</read_first>

<action>
Search `packages/runtime/src/runtime.ts` and `packages/runtime/src/event-buffer.ts` for all calls to `enforce(`. At each call site, the original NIP-01 message array (`msg`) is in scope. Pass it as the third argument.

Pattern — change each occurrence from:
```typescript
const result = enforce(pubkey, capability);
```
to:
```typescript
const result = enforce(pubkey, capability, msg);
```

Where `msg` is the raw NIP-01 message array available in scope at each dispatch handler. If a call site uses a different variable name for the message (e.g., `message`, `raw`), use that variable instead.

For `event-buffer.ts` line 108, enforce is called for recipient-side delivery checks inside `deliverToSubscriptions(event, senderId)`. The original raw NIP-01 message is not in scope (only the `event: NostrEvent` object). Two options:
- Pass `undefined` (acceptable — recipient-side checks are informational, not user-initiated denials)
- Construct a synthetic message: `['EVENT', event]`

**Use the synthetic approach** — pass `['EVENT', event]` so the AclCheckEvent has useful message context even for delivery-time checks. Change line 108 from:
```typescript
const recipientResult = enforce(recipientPubkey, 'relay:read');
```
to:
```typescript
const recipientResult = enforce(recipientPubkey, 'relay:read', ['EVENT', event]);
```

Also update the `enforce` parameter type in `createEventBuffer` function signature (line 92) from:
```typescript
enforce: (pubkey: string, capability: Capability) => EnforceResult,
```
to:
```typescript
enforce: (pubkey: string, capability: Capability, message?: unknown[]) => EnforceResult,
```

Do NOT change any call sites in test files — tests can continue passing `undefined` for message (since it's optional).
</action>

Create `apps/demo/src/acl-history.ts` with:

1. **Extended event type:**
```typescript
import type { AclCheckEvent } from '@napplet/shell';

export interface AclHistoryEntry extends AclCheckEvent {
  /** Monotonic index for ordering. */
  index: number;
  /** Timestamp when the event was captured. */
  timestamp: number;
  /** The windowId of the napplet (resolved from pubkey). */
  windowId: string;
  /** Human-readable napplet name (for display). */
  nappletName: string;
}
```

2. **Ring buffer store:**
```typescript
/** Default ring buffer size per napplet. */
const DEFAULT_ACL_RING_SIZE = 50;

let aclRingSize = DEFAULT_ACL_RING_SIZE;

/** Set the ACL ring buffer size (called from demo-config integration). */
export function setAclRingSize(size: number): void {
  aclRingSize = Math.max(1, Math.floor(size));
}

/** Get current ring buffer size. */
export function getAclRingSize(): number {
  return aclRingSize;
}

let nextIndex = 0;

// Per-windowId ring buffers
const aclRings = new Map<string, AclHistoryEntry[]>();

// Global ring (all napplets combined, for ACL node view)
const globalAclRing: AclHistoryEntry[] = [];
```

3. **Push function:**
```typescript
export function pushAclEvent(
  event: AclCheckEvent,
  windowId: string,
  nappletName: string,
): void {
  const entry: AclHistoryEntry = {
    ...event,
    index: nextIndex++,
    timestamp: Date.now(),
    windowId,
    nappletName,
  };

  // Per-napplet ring
  let ring = aclRings.get(windowId);
  if (!ring) {
    ring = [];
    aclRings.set(windowId, ring);
  }
  ring.push(entry);
  if (ring.length > aclRingSize) ring.shift();

  // Global ring
  globalAclRing.push(entry);
  if (globalAclRing.length > aclRingSize * 4) globalAclRing.shift();
}
```

4. **Query functions:**
```typescript
/** Get ACL history for a specific napplet. */
export function getAclHistory(windowId: string): AclHistoryEntry[] {
  return aclRings.get(windowId) ?? [];
}

/** Get denial-only history for a specific napplet. */
export function getAclDenials(windowId: string): AclHistoryEntry[] {
  return (aclRings.get(windowId) ?? []).filter(e => e.decision === 'deny');
}

/** Get global ACL history (all napplets). */
export function getGlobalAclHistory(): AclHistoryEntry[] {
  return globalAclRing;
}

/** Get global denial-only history. */
export function getGlobalAclDenials(): AclHistoryEntry[] {
  return globalAclRing.filter(e => e.decision === 'deny');
}

/** Clear all history (for testing/reset). */
export function clearAclHistory(): void {
  aclRings.clear();
  globalAclRing.length = 0;
  nextIndex = 0;
}
```
</action>

    onAclCheck: (event) => {
      // Resolve windowId and name from pubkey
      let windowId = '';
      let nappletName = 'unknown';
      for (const [wid, info] of napplets) {
        if (info.pubkey === event.identity.pubkey) {
