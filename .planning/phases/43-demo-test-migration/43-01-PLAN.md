# Plan 43-01: Migrate Demo Napplets to SDK Imports

## Objective

Update chat and bot demo napplets from old named-export shim imports (`import { publish, subscribe, emit, on, nappState } from '@napplet/shim'`) to SDK imports (`import { relay, ipc, storage } from '@napplet/sdk'`) plus side-effect shim import (`import '@napplet/shim'`). Demo napplets are teaching tools — they use the recommended SDK developer path per CONTEXT decision D-01.

7. `subscribe([{ kinds: [1], ... }], ...)` → `relay.subscribe([{ kinds: [1], ... }], ...)` (line ~152 after AUTH)

Also update the file header comment to reflect the new API:

- Change "subscribe()" → "relay.subscribe()"

```typescript

import path from 'path';
import { nip5aManifest } from '@napplet/vite-plugin';

  base: './',
  plugins: [
    nip5aManifest({
      nappletType: 'demo-chat',
    }),
  ],
  resolve: {
    alias: {
      '@napplet/shim': path.resolve(__dirname, '../../../../packages/shim/src/index.ts'),
      '@napplet/sdk': path.resolve(__dirname, '../../../../packages/sdk/src/index.ts'),
    },
  },
  build: {
    outDir: 'dist',
    emptyDirBeforeWrite: true,
  },
});
```
