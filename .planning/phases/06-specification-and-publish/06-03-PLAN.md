  # @napplet/vite-plugin

  > Vite plugin for napplet local development — injects aggregate hash meta tags and optionally generates NIP-5A manifests for testing.

  **This is a development tool.** For production deployment of napplets to nsites, use community deploy tools like [nsyte](https://github.com/nicefarm/nsyte) which handle NIP-5A event creation and relay publishing.

  ## Getting Started

  [Explain what the plugin does during development:
  1. In dev mode: injects an empty `<meta name="napplet-aggregate-hash">` tag so the shim can find it
  2. At build time (with VITE_DEV_PRIVKEY_HEX set): computes per-file SHA-256 hashes, aggregate hash, and writes a .nip5a-manifest.json for local testing
  The build-time manifest is for verifying the hash computation workflow locally, NOT for deploying to relays.]

  ### When to Use This
  - You are building a napplet and testing locally with @napplet/shell
  - You want to verify aggregate hash computation before deploying

  ### When NOT to Use This
  - Deploying napplets to production (use nsyte or similar)
  - Creating NIP-5A events for relay publishing (use dedicated tools)

  ## Installation

  ```bash
  npm install -D @napplet/vite-plugin
  ```

  Note: This is a devDependency. It is not needed at runtime.

  ## Quick Start

  [Minimal vite.config.ts example]

  ## Configuration

  ### Plugin Options

  #### nappType (required)
  [Type: string. The napp type identifier, e.g., 'feed', 'chat'. Used in the napplet-napp-type meta tag.]

  ### Environment Variables

  #### VITE_DEV_PRIVKEY_HEX
  [Hex-encoded 32-byte private key for signing test manifests. If not set, build-time manifest generation is skipped (graceful no-op). NEVER use a real private key here — use a dedicated test key.]

  ## How It Works

  ### Dev Mode (transformIndexHtml)
  [Injects `<meta name="napplet-aggregate-hash" content="">` and `<meta name="napplet-napp-type" content="...">` into HTML head]

  ### Build Mode (closeBundle)
  [Only runs if VITE_DEV_PRIVKEY_HEX is set:
  1. Walks dist/ directory

  6. Signs with test private key
  7. Writes .nip5a-manifest.json to dist/
  8. Updates meta tag in dist/index.html with computed hash]
