<task id="61-01-02">
<title>Replace NIP-style setext header with internal reference header</title>
<read_first>
- RUNTIME-SPEC.md (first 30 lines — the NIP-XXXX setext header and subtitle)
</read_first>
<action>
Replace the first ~8 lines of RUNTIME-SPEC.md (the NIP-style setext header) with a standard ATX heading and internal reference notice. The current header is:

```markdown
NIP-XXXX
========

Napplet Shell Protocol: Iframe-to-Shell Communication via postMessage
---------------------------------------------------------------------

`draft` `optional`
```

Replace it with:

```markdown
# Napplet Shell Protocol - Runtime Reference

> **Internal documentation.** This is the detailed runtime implementation reference
> for the napplet protocol. For the protocol standard, see
> [NIP-5D](https://github.com/nostr-protocol/nips/blob/master/5D.md).

---
```

Remove the `## Status` section (lines ~10-11, which says "This is a draft specification...") since the NIP-5D file is now the draft specification. The rest of the document (starting from `## Referenced NIPs`) should remain unchanged.
</action>
<acceptance_criteria>
- `head -1 RUNTIME-SPEC.md` outputs `# Napplet Shell Protocol - Runtime Reference`
- `grep "Internal documentation" RUNTIME-SPEC.md` returns a match
- `grep "NIP-5D" RUNTIME-SPEC.md` returns at least one match (the link in the header)
- `grep -c "NIP-XXXX" RUNTIME-SPEC.md` returns 0 (old header removed)
- `grep -c "========" RUNTIME-SPEC.md` returns 0 (setext underline removed)
- The file does NOT contain a `## Status` section with "This is a draft specification"
- `grep "## Referenced NIPs" RUNTIME-SPEC.md` still returns a match (rest of content preserved)
</acceptance_criteria>
</task>
